# Code of Conduct

MineIA follows the Contributor Covenant Code of Conduct.

## Standards

Participants are expected to:

- use respectful language
- accept constructive feedback
- focus discussions on the project
- avoid harassment, insults and personal attacks
- respect privacy and security boundaries

## Enforcement

Maintainers may remove comments, close issues or restrict participation when behavior harms the project or community.

## Reporting

Report conduct issues through the repository maintainers. Include relevant links, screenshots or context when possible.
